# Security policy

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email [roddarjohn@protonmail.com](mailto:roddarjohn@protonmail.com) with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours. If you do not, follow up to confirm the message was received.

Please give us reasonable time to address the issue before any public disclosure.
